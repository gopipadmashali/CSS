The CSS Box Model

What is the Box Model?

The CSS box model consists of four parts: the content, padding, border, and margin. It defines how elements are structured and spaced on a webpage.

Content-Box Model

This box uses the content-box model, where padding and border are added to the width and height of the element.

Border-Box Model

This box uses the border-box model, where padding and border are included in the total width and height of the element.

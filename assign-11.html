Static Positioning: This is the default position. Elements appear in the normal document flow.
Relative Positioning: This element is moved relative to its normal position.
Absolute Positioning: This element is positioned relative to the nearest positioned ancestor or the document if none exists.
Fixed Positioning: This element is fixed relative to the viewport. It stays in place even when scrolling.
Sticky Positioning: This element becomes fixed at the top when you scroll past it.
